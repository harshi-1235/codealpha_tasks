Cloud Bus Booking System
Description
The Cloud Bus Booking System is a Python-based desktop application developed using Tkinter for the graphical user interface and SQLite as the database. The system allows users to register, log in, book bus tickets, view their bookings, and cancel bookings. It also includes an admin panel for managing bus records.
Features:
 User Registration
 User Login
 View Available Buses
 Book Bus Tickets
 View My Bookings
 Cancel Bookings
 Admin Panel
 SQLite Database Integration
 Simple and User-Friendly Interface
Technologies Used:
 Python
 Tkinter
 SQLite3
