# 🚌 Cloud Bus Booking System

## 📌 Project Overview
The **Cloud Bus Booking System** is a Python-based desktop application developed using **Tkinter** for the graphical user interface and **SQLite3** as the backend database.

This system provides a complete bus reservation workflow including user authentication, ticket booking, booking management, and an admin panel for managing bus details.

The application is designed to simulate a real-world booking system with a simple, clean, and user-friendly interface.

---

## 🎯 Objective
The objective of this project is to develop a desktop-based bus reservation system that demonstrates practical implementation of:

- User authentication system  
- Booking management operations  
- Database-driven application design  
- GUI development using Python
