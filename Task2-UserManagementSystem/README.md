# 🧾 User Management System (Tkinter + SQLite)

## 📌 Project Overview
The **User Management System** is a desktop-based application developed using Python.  
It provides a simple and interactive graphical user interface (GUI) using **Tkinter** and uses **SQLite** as the backend database.

This project is designed to perform complete **CRUD operations** (Create, Read, Update, Delete) with proper validation and duplicate prevention, making it efficient for basic data management tasks.

---

## 🎯 Objective
The main objective of this project is to demonstrate how a Python GUI application can interact with a database to manage user records efficiently while maintaining data integrity and providing a user-friendly interface.

---

## ✨ Key Features

✔ Add new user records (Name, Email, Phone)  
✔ View all stored records in a separate window  
✔ Search records using email  
✔ Update existing user details  
✔ Delete user records  
✔ Duplicate prevention (Email & Phone validation)  
✔ Simple and clean GUI design
